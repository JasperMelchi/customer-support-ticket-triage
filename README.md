# customer-support-ticket-triage
Customer Support Ticket Triage System

This project is developed as part of the Week-1 Internship assignment.
It demonstrates how customer support tickets can be analyzed, classified,
and prioritized using Python.

Project Features:
- Load and explore customer support ticket data
- Clean and preprocess ticket messages
- Classify issues into categories like PAYMENT, LOGIN, DELIVERY, etc.
- Assign priority levels (P0â€“P3) based on urgency
